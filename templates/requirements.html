System Requirements

To ensure CS50 Switch works properly, your setup must meet the following hardware, software, and network requirements. The system has been built and tested with a very specific configuration, and it's currently only compatible with that environment.

Hardware Requirements

  • Microcontroller: ESP32-WROOM-32 (or equivalent with integrated Wi-Fi)
  • Servo Motor: SG90 micro servo or MG90S (PWM-controlled)
  • Power Source: USB 5V with stable current (recommended â‰¥1A)
  • Mounting Method: Tape or 3D-printed arm for holding servo in position
  • Computer: Any device capable of running Python and Flask locally

Software Requirements

  • Python 3.10+ (for backend Flask app)
  • Flask web framework
  • Arduino IDE (to upload ESP32 firmware)
  • Arduino Libraries: WiFi, WebServer, ESP32Servo, time.h
  • Flask dependencies: requests, sqlite3 (included in Python stdlib)

Network Requirements

  • Wi-Fi Network: Must use the same network the ESP32 is connected to
  • SSID: Hardcoded to FABULAB in the ESP32 sketch
  • Password: Must match the ESP32 sketch (ROUTERCASA01)
  • IP Configuration: You must know your ESP32's local IP address (e.g., 192.168.0.38)
  • Important: Not compatible with networks that change the IP on reboot (unless IP reservation is used)

Compatibility Notes

CS50 Switch has been designed and tested for one specific ESP32 board and one Wi-Fi network. If you change the network name (SSID) or use a different ESP32 model, the current firmware and configuration may not work.

If you wish to adapt the system to your own environment, you will need to:

  • Edit the ESP32 sketch to match your Wi-Fi credentials
  • Update the IP address in your Flask backend accordingly
  • Ensure your router provides consistent IP (via DHCP reservation or static IP)

Important Disclaimer

CS50 Switch is not intended for integration with real home automation or high-voltage systems. It is a prototype built for safe, low-voltage servo control only. Do not attempt to connect this system to high-current circuits or control electrical power directly without proper certification and supervision.
