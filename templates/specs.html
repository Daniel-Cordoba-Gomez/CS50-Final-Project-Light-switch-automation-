Technical Specifications

CS50 Switch is designed to be simple to implement, flexible for experimentation, and optimized for working within your local network. Below you’ll find a full breakdown of the hardware and software components, electrical characteristics, and operational parameters for the system.

Hardware Components

  • Microcontroller: ESP32-WROOM-32 (or similar ESP32 board with Wi-Fi)
  • Servo Motor: SG90 or MG90S 9g servo
  • Power Supply: 5V USB or external power adapter (with enough current for servo)
  • Control Pin: GPIO13 (used for controlling the servo rotation)
  • Mounting: Double-sided tape, adhesive mount, or 3D printed holder for light switch

Software Stack

  • Backend: Flask (Python) with session-based login
  • Frontend: Jinja templating with custom CSS and responsive design
  • Database: SQLite (used for user login/sign-up)
  • ESP32 Code: Arduino framework with WebServer library for handling HTTP POST requests
  • Communication Protocol: Local HTTP POST requests via Wi-Fi (no internet/cloud required)

Servo Operation

  • ON Command: Servo rotates to 90° to press light switch ON
  • OFF Command: Servo rotates back to 0° to reset to OFF
  • Rotation Delay: 2 seconds to simulate realistic switch press
  • Initial Position: 0° (resting)

Environmental Conditions

  • Recommended indoor use only
  • Temperature range: 0°C to 40°C (non-condensing)
  • Placement: Close to light switch (distance dictated by servo cable length)

Web Access & Compatibility

  • Accessible via local network only (e.g., http://192.168.0.38)
  • Compatible with most modern browsers (Chrome, Firefox, Safari, Edge)
  • Mobile and tablet-friendly interface

Security

  • Login required to use alarm and switch controls
  • Alarm and switch actions restricted to authenticated sessions
  • No external network exposure (stays on LAN)

Educational Use Only

CS50 Switch is an academic project built for Harvard’s CS50 final project. It is not a commercial product, and not certified for household electrical automation. Please use it responsibly and only for safe, low-voltage servo-based control.
